**A - Wrong Answer**

- 문제 설명
  
  $0$과 $9$ 사이 양의 정수 $A$와 $B$가 주어지고, $A+B$가 아닌 수를 $0$에서 $9$사이 아무 수나 출력하면 됩니다.
  
- 해설
  
  $(A+B+1)\%10$을 출력하면 언제나 $A+B$가 아닌 수를 출력할 수 있습니다. 
  
- 코드
  ```cpp
  int main() {
      ios::sync_with_stdio(0), cin.tie(0);
      int a, b; cin >> a >> b;
      cout << ((a+b)+1)%10 <<'\n';
  }
  ```
